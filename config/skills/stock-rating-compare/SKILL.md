# Skill: stock-rating-compare

查詢台股ETF成分股 vs 投顧近月目標價對照表.

## 用途

當使用者需要以下資訊時啟動:
- "0050成分股最近有哪些投顧調整目標價?"
- "查台積電/聯發科最近的外資目標價"
- "整理近一個月投顧對0050的評等變動"
- "哪幾檔0050成分股有投顧調升目標價?"
- "比較0050各成分股當前股價 vs 投顧目標價"

## 輸入參數

| 參數 | 說明 | 預設值 |
|:-----|:-----|:-------|
| etf_code | ETF代號 (0050/0056/006208等) | 0050 |
| months | 查詢回溯月份數 | 1 |
| stock_filter | 指定股票代號(逗號分隔), 不指定則全部 | 全部成分股 |
| output | 輸出格式: table / json | table |
| source | 資料來源: auto / cnyes / news | auto |

## 核心流程

### Step 1: 取得ETF成分股 + 當前股價

使用 `scripts/get_etf_data.py` 從 etfinfo.tw 抓取:
- 完整成分股代號/名稱/權重
- 最新收盤價/漲跌幅
- 快照日期 (通常為昨日收盤)

```
python3 scripts/get_etf_data.py 0050
```

**技術細節**: etfinfo.tw 使用 Nuxt SSR, 資料嵌入在 `<script id="__NUXT_DATA__">` 中. 腳本自動解析序列化格式, 提取 holdings + prices.

### Step 2: 查詢鉅亨網外資評等

使用 `scripts/search_cnyes_ratings.py` 掃描鉅亨網外資評等頁面:
- 指定目標股票代號清單 (從Step 1取得)
- 掃描前 N 頁 (依月份數調整)
- 提取: 股票代號、目標價、券商名稱、調整方向、日期

```
python3 scripts/search_cnyes_ratings.py --codes 2330,2454,2308 --pages 5
```

### Step 3: 搜尋新聞補充國內投顧資料

使用 websearch 搜尋以下關鍵字補齊國內投顧資料:
- `"{stock_name} 目標價 {month}"` (如 "台積電 目標價 5月")
- `"{投顧名稱} 研究報告 {stock_name}"` (如 "統一投顧 台光電")
- `"高盛 {stock_name} 目標價"` (如 "高盛 台積電 目標價")
- `"{ETF代碼} 成分股 投顧 評等 {month}"`

重點搜尋投顧: 高盛、摩根士丹利、花旗、摩根大通、野村、統一投顧、元大投顧、中信投顧、富邦投顧、凱基投顧、群益投顧

### Step 4: 交叉比對 + 產出表格

合併以上三步資料, 產出結構化對照表:

```
# 格式範例
| 代號 | 名稱 | 權重% | 當前股價 | 投顧 | 目標價 | 調整 | 日期 |
```

**比對邏輯**:
- 有評等變動的股票列在表格上半部
- 無評等變動的列在表格下半部 (只顯示代號/名稱/股價)
- 當前股價 vs 目標價差距用顏色標記 (漲/跌/合理)
- 國內投顧 vs 外資目標價差異大時特別標註

## 資料來源

| 來源 | 類型 | 說明 |
|:-----|:-----|:-----|
| etfinfo.tw | ETF持股+股價 | 快照資料, Nuxt SSR embedded JSON |
| 鉅亨網 外資評等 | 外資目標價調整 | Factset數據, 約12頁, 每日更新 |
| 新聞搜尋(websearch) | 國內投顧報告摘要 | 統一/元大/中信/富邦/凱基/群益等 |
| 元大投信官網 | 官方持股明細 | yuantaetfs.com, 每月更新 |

## 限制與注意事項

1. **國內投顧完整報告需付費**: 僅能從新聞摘要取得部分資訊, 可能不完整
2. **鉅亨網頁面動態載入**: 部分數據透過JS渲染, 簡單fetch可能拿不全
3. **目標價時效性**: 發布日 vs 快照日可能差距數週, 需標註日期
4. **0050權重變動**: 每季調整成分股(3/6/9/12月), 快照日期很重要
5. **股價比較非建議**: 當前股價 vs 目標價僅供參考, 不構成投資建議

## 腳本說明

`scripts/get_etf_data.py`:
- 無外部套件依賴 (只用 stdlib)
- 輸出JSON可直接 pipe 給其他工具
- 支援任意台股ETF (不限定0050)

`scripts/search_cnyes_ratings.py`:
- 無外部套件依賴
- --codes 指定目標股票
- --pages 指定掃描頁數

## 常見調整

- **ETF代號**: 006208 (富邦台50), 0056 (元大高股息), 00878 (國泰永續高股息)
- **時間範圍**: 若使用者說"近一週", 設定 `--pages 3`; "近三個月" 設 `--pages 15`
- **指定股票**: 如只想看台積電+聯發科, 用 `--codes 2330,2454`
